# Todo Vue

Простой SPA‑проект для управления задачами, созданный на Vue 3 (+ Composition API) и собранный с помощью Vite.

Кратко: приложение позволяет добавлять, редактировать, удалять и отмечать задачи как завершённые, искать задачи и переключать тему интерфейса.


## Технологический стек

- Vue 3 (Composition API)
- Vite — сборка и запуск проекта
- SASS (SCSS) — оформление интерфейса
- JavaScript 


## Ключевые реализации и концепции Vue

- Используется Composition API (setup, ref, reactive).
- Передача данных между компонентами реализована через `props` и `emit`.
- Реактивное состояние: `ref`, `reactive`.
- Вычисляемые значения через `computed`, реакция на изменения через `watch`.
- Модальное окно реализовано через условный рендеринг (`v-if`).
- Основные операции: добавление, редактирование, удаление, отметка как завершённая.
- Есть переключатель темы — может быть реализован через Tailwind-классы или локальное состояние Vue.
- Поиск по задачам реализован в отдельном компоненте.


## Архитектура проекта (важные файлы и папки)

- `index.html` — точка входа приложения
- `src/`
  - `main.js` — инициализация приложения
  - `App.vue` — корневой компонент
  - `components/` — каталог компонент
    - `TodoItem.vue` — компонент отдельной задачи
    - `TodoList.vue` — компонент списка задач
    - `AddTodoModal.vue` — модальное окно / форма добавления задачи
    - `SearchBar.vue` — поиск задач
    - `ThemeToggle.vue` — переключатель темы
  - `assets/` — статические ресурсы (иконки, изображения)
- `scss/` — стили проекта (структуризация по блокам: base, components, layout, themes, utils)


## Фичи

- Добавление новых задач
- Редактирование существующих задач
- Удаление задач
- Отметка задачи как выполненной
- Поиск и фильтрация задач
- Переключение темы интерфейса (светлая / тёмная)
- Модальное окно для создания/редактирования задач
